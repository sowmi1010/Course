MyDocs – Interactive Learning Dashboard
MyDocs is an interactive learning platform where users can explore programming topics, view examples, and manage favorites. It’s built with React, Tailwind CSS, and Monaco Editor for an amazing coding experience.

Features
Dashboard – Quick access to multiple programming languages
Topic List – Search, sort, and favorite topics
Topic Editor – Add/Edit topics with code snippets & subtopics
Code Preview – Live preview for HTML/CSS/JS
Favorites – Save your favorite topics for quick access
Responsive Design – Works on mobile & desktop

Tech Stack
Frontend: React.js, Tailwind CSS
Code Editor: Monaco Editor
Icons: React Icons
Styling: Tailwind + Custom Components
API: Axios
Routing: React Router

Installation
Clone the repository and install dependencies:

git clone https://github.com/sowmi1010/Documentation.git
cd mydocs-app
cd frontend:
npm install
npm run dev
cd backend:
npm install
nodemon server.js

Navigate to Dashboard: 
Select a language → View/Add/Edit topics
Preview code in real-time
Add favorites for quick access

Authentication:
User login/logout is handled with AuthContext
